# SAT-Based Sudoku Solver

C++ example using MiniSat to solve Sudoku. From the blog series at https://codingnest.com/modern-sat-solvers-fast-neat-underused-part-1-of-n/

## What is in this repo

- C++ code encoding Sudoku as SAT and solving with MiniSat.
- Build and run instructions.

## How to run

Install MiniSat and a C++ compiler, build the project, run the executable. See repo for puzzle format.
